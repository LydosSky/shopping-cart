# Shopping Cart

A mock shopping cart application using the FakeStore API to display store items.

## Overview

A mock shopping cart application that fetches product data from the FakeStore API and allows users to browse, search, and sort items while managing a client-side cart using a modern React interface.

## Features

* Product browsing, search, and sorting
* Client-side shopping cart
* Responsive React-based UI

## Tech Stack

* React

## Getting Started

### Prerequisites

* Node.js
* Package manager (npm)

### Installation

```bash
# clone the repository
git clone https://github.com/LydosSky/shopping-cart.git
cd shopping-cart

# install dependencies
npm install
```

### Run Locally

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or configured port).

## Usage

Once the application is running, users can browse products, search and sort items, add products to the cart, and adjust quantities directly from the interface.
